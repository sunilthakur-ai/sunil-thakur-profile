# Assets Folder

This folder contains images and media files for the website.

## Profile Picture

Current profile picture: `Sunil_Thakur_v1.jpg`

To update your profile picture in the future:
1. Add new version with versioned name: `Sunil_Thakur_v2.jpg`
2. Update `index.html` line ~66 to point to new version:
   ```html
   <img src="assets/Sunil_Thakur_v2.jpg"
   ```
3. Commit and push changes
4. Website updates automatically in 1-2 minutes

## Image Specifications

**Profile Picture:**
- Format: JPG or PNG
- Recommended size: 500x500 pixels (square)
- Max file size: 500KB for fast loading
- Must be square for best results

## Future Assets

## Versioning

Use versioning in filenames to maintain history:
- `Sunil_Thakur_v1.jpg`
- `Sunil_Thakur_v2.jpg`
- `project_screenshot_v1.png`

This makes it easy to roll back or reference previous versions.
